# ScaleX - Insights & Blogs

A modern, dark-themed static webpage displaying the **ScaleX** blog and insights section.

## Features

- **Dark Theme Interface**: Clean design with a black background and white text.
- **Flexbox Layout**: Uses CSS Flexbox for structured and responsive component alignment.
- **Navigation Bar**: Top header with a logo, navigation links, and a "Get Started" button.
- **Category Filters**: Pill-shaped filter buttons (All, Tech, SaaS, Marketing).
- **Blog Cards**: Individual cards featuring a thumbnail image, category tag, title, and published date.

## Technologies Used

- **HTML5**: Page structure and content.
- **CSS3**: Custom styling, Flexbox layout, and typography (Google Fonts Roboto).
- **JavaScript**: Blank `script.js` file included for adding future interactive capabilities.

## Getting Started

To run this project locally:

1. Clone or download the repository to your local machine.
2. Ensure the file structure remains intact, particularly the `assets` folder containing images.
3. Open `index.html` in your preferred web browser. Alternatively, use a tool like VS Code's "Live Server" extension for a better development experience.
