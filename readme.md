# ScaleX - Insights & Blogs

A modern, dark-themed static webpage displaying the **ScaleX** blog and insights section.

## 🚀 Features

- **Dark Theme Interface**: Clean, modern aesthetic with a black background and high-contrast white text.
- **Flexbox Layout**: Structured and flexible component alignment using CSS Flexbox.
- **Navigation Bar**: Top header featuring the brand logo, navigation links, and a prominent "Get Started" call-to-action button.
- **Category Filters**: Interactive pill-shaped filter buttons (All, Tech, SaaS, Marketing).
- **Blog Cards**: Visually appealing cards detailing a thumbnail image, category tag, title, and published date.

## 🛠️ Technologies Used

- **HTML5**: Semantic page structure and content mapping.
- **CSS3**: Custom styling, Flexbox layout, hover effects, and typography (Google Fonts 'Roboto').
- **JavaScript**: A linked `script.js` file ready for future interactive capabilities (e.g., filtering).

## 🏁 Getting Started

To run this project locally:

1. Clone or download the repository to your local machine.
2. Ensure the file structure remains intact, particularly the `assets` folder containing the images.
3. Open `index.html` in your preferred web browser.
   _(Alternatively, use a tool like VS Code's "Live Server" extension for a better development experience with hot reloading)._
