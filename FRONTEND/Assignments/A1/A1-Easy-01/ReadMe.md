# Pokémon Gen I Showcase

A sleek, visually appealing frontend web interface showcasing Generation I Pokémon. The current layout features Pikachu with a modern, split-screen background design.

## 🚀 Features

- **Split-Screen UI:** A vibrant two-tone background (yellow and light beige) that beautifully frames the featured Pokémon.
- **Responsive Elements:** Built with Flexbox and percentage-based widths for a fluid layout.
- **Iconography:** Integrated with [Remix Icon](https://remixicon.com/) for lightweight and clean navigation icons.
- **Interactive Potential:** Includes navigation dots (indicators) and an empty `script.js` ready for adding image-slider functionality to cycle through Bulbasaur, Charizard, and Mewtwo.

## 🏃‍♂️ How to Run

1. Clone or download this repository to your local machine.
2. Navigate to the project directory.
3. Simply open `index.html` in your preferred web browser, or use a tool like **Live Server** in VS Code for a better development experience.
