# Art Work Gallery

A responsive and visually appealing front-end project showcasing a gallery of digital artworks. The interface features a dark gradient theme with glassmorphism-inspired card designs for an elegant presentation.

## Features

- **Responsive Grid Layout:** Utilizes CSS Flexbox to organize artwork cards neatly.
- **Modern UI Design:** Includes a vibrant background gradient, semi-transparent artwork cards, and hover interactions.
- **Custom Assets & Icons:** Displays high-quality images and uses Remix Icons for intuitive UI elements like the "like" button.

## Project Structure

```text
├── assets/           # Contains all images and the favicon
├── index.html        # Main HTML layout and structure
├── Readme.md         # Project documentation
├── script.js         # Placeholder for future JavaScript functionality
└── style.css         # Styling and responsive design rules
```

## Technologies Used

- **HTML5**
- **CSS3**

## Getting Started

1. Clone or download the project files.
2. Open `index.html` in your preferred web browser to view the application. No local development server is strictly required, but using one (like VS Code's "Live Server" extension) is recommended.
