# 🚀 Productivity Dashboard

A modern, responsive, and feature-rich **Productivity Dashboard** built using **HTML, CSS, and JavaScript**. It brings together essential productivity tools in a single interface, helping users organize their day, manage tasks, set goals, and stay focused.

---

## ✨ Features

### 📅 Dashboard
- Real-time date and time
- Dynamic greeting based on time of day
- Responsive dashboard layout
- Theme switching (Light/Dark)

### 🌤️ Weather
- Current weather information
- Location-based weather updates
- Dynamic weather icons and background

### ✅ Task Manager
- Add and manage daily tasks
- Mark important tasks
- Delete completed tasks
- Local storage support

### 🎯 Goal Tracker
- Create and organize personal goals
- Responsive goal cards
- Delete completed goals
- Persistent storage

### 📝 Notes
- Create quick notes
- Store notes locally
- Clean and simple interface

### 🎵 Focus Mode
- Background ambience/music support
- Designed to improve concentration

### 🎨 Modern UI
- Responsive design
- Smooth animations
- Glassmorphism-inspired interface
- Interactive cards and transitions

---

## 🛠️ Tech Stack

- HTML5

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/productivity-dashboard.git
```

### Navigate to the project

```bash
cd productivity-dashboard
```

### Open the project

Simply open **index.html** in your browser.

No additional installation is required.
